# Wallace

Wayland layer shell version of Gromit-MPX.

## Usage

- Mouse left: draw stuff
- Mouse right: clear drawing
- Mouse wheel: change color
- `pkill -SIGUSR1 wallace`: toggle bottom/overlay layer
